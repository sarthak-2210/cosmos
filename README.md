# COSMOS — Solar System Simulator

Interactive 3D solar system built with **Three.js** and **Vite**.

## Live demo

**https://sarthak-2210.github.io/cosmos/**

## Features

- Real-time orbital simulation of the Sun and planets
- Body focus camera and info panel
- Adjustable simulation speed
- Starfield background and modern HUD UI

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build

```bash
npm run build
npm run preview
```

## Deploy

Pushes to `main` deploy automatically to GitHub Pages via GitHub Actions.
